# 주요기능
### 게시글 기능
1. 글쓰기
2. 글목록
3. 글조회
4. 글수정
5. 글삭제
6. 페이징처리
7. 파일(이미지) 다중 첨부


### 댓글 기능(Ajax)
1. 글 상세 페이지에서 댓글 입력
2. 상세조회할 때 기존에 작성된 댓글 목록이 보임
3. 댓글을 입력하면 기존 댓글 목록에 새로 작성한 댓글 추가
